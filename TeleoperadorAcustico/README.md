
# WebSim repository

## Teleoperator drone


| Microphone  	| Function                	|
|-------------	|-------------------------	|
| go           	| Moves drone frontwards  	|
| seven         | Moves drone backwards   	|
| stop          | The   drone stops     	  |

----------------------------------------------
| left          | Turn left                 |
| right        	| Turn right              	|
| up          	| Go up drone             	|
| down         	| Do down drone            	|
