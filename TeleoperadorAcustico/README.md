
# WebSim repository

## Teleoperator drone


| Microphone  	| Function                	|
|-------------	|-------------------------	|
| Go           	| Moves drone frontwards  	|
| Back          | Moves drone backwards   	|
| Stop          | The   drone stops     	  |
| Left          | Turn left                 |
| Right        	| Turn right              	|
| Up          	| Go up drone             	|
| Down         	| Do down drone            	|
